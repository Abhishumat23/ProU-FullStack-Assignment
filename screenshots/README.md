## Taking Screenshots

1. Run both backend and frontend servers
2. Login with demo credentials
3. Navigate to each page
4. Take full-page screenshots
5. Save with descriptive names
6. Update README.md to reference these screenshots

## Screenshot Guidelines

- Use high resolution (at least 1920x1080)
- Show meaningful data (use the seeded demo data)
- Include browser chrome to show the URL
- Highlight key features with annotations if needed
- Ensure no sensitive information is visible

## Quick Screenshot Commands

### macOS
```bash
# Full screen: Cmd + Shift + 3
# Selected area: Cmd + Shift + 4
```

### Windows
```bash
# Full screen: PrtScn
# Active window: Alt + PrtScn
# Snipping Tool: Windows + Shift + S
```

### Linux
```bash
# Use gnome-screenshot or similar tool
gnome-screenshot -w  # Current window
gnome-screenshot -a  # Selected area
```
